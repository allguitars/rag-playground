# rag-playground

This repository is a playground for experimenting with RAG, LangChain, and LangGraph.

- `langchain/`：LangChain-based RAG and chain examples
- `langgraph/`：State-machine and agentic flow examples
- `observability/`：Tracing and evaluation demos (e.g. LangSmith)
- `examples/`：Usage and UI examples (e.g. Gradio)
- `data/raw/`：Source documents fed into RAG
- `data/vectorstore/`：Local vector stores (rebuildable)

Notebooks are for experimentation; stable logic may later be migrated to `.py`.
