# Prompts

This directory contains prompt templates used across RAG and agent experiments.

## Guidelines

- Store prompts as plain text or markdown.
- Avoid hardcoding prompts directly in notebooks.
- Name prompts based on intent, not model (e.g. `rag_system.md`, not `gpt4_prompt.md`).
- Prompts should be reusable across LangChain and LangGraph implementations.

## Example

- `rag_system.md`
- `rag_qa.md`
